# WiFi Customer Renewal Management System

A production-ready WiFi Customer Renewal Management Mobile Application built with React Native, Node.js, and PostgreSQL.

## Project Overview

Automatically monitor customer internet plan expiry dates and send reminders through WhatsApp, SMS, and Push Notifications.

## Tech Stack

### Mobile
- React Native
- TypeScript
- Redux Toolkit
- React Query
- React Navigation

### Backend
- Node.js
- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL

### Notifications
- WhatsApp Business Cloud API
- Firebase Cloud Messaging
- SMS Gateway (MSG91, Twilio)
